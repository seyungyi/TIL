### 압축풀기 관련

```bash
# tar [[-]function][options] filenames
# tar [[-]function][options] -C directory-name

function : 수행할 작업을 지시하는 단일 명령
option : 처리할 기능
filenames : 풀거나 묶을 파일 리스트(목록)

tar(Type Archiver)
function
c : 묶음
x : 파일 추출(풀기)
d : 파일과 비교
r : 파일의 마지막 부분에 파일 추가
t : 내용 나열
u : 새로운 파일로 업데이트

option
f : 읽거나 기록할 파일을 정의
v : 묶거나 풀고 있는 파일을 보기
z : gzip으로 압축하여 백업

압축하기
# tar cvf file_name.tar *.c
# gzip file_name.tar

압출풀기
# tar xvzf file_name.tar
# unzip file_name.zip
```
